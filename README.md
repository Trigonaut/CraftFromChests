<h1>CraftFromChests</h1>

Allumeria mod to allow crafting using items from nearby chests

<h2>To play with CFC</h2>

1. Install the [Ignitron mod loader](https://github.com/danilwhale/ignitron).<br/>
2. [Download CFC](https://github.com/Trigonaut/CraftFromChests/releases), extract it, and put the `CraftFromChests` folder in your `Allumeria/mods` directory, alongside `Loader.dll`<br/><br/>

The directory should look something like this:
```bash
Allumeria
├── Allumeria.exe
└── mods
    ├── 0Harmony.dll
    ├── Loader.dll
    └── CraftFromChests
        ├── CraftFromChests.dll
        └── metadata.json
```
